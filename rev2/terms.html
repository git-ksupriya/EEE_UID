RL - based Frequency Control for Renewable Energy Microgrids: TERMS

These are the Key Components for the PV Grid Model:

PV Array:
This is your solar panel. It takes sunlight (represented by the input 'Ir' which stands for irradiance) and converts it into DC (Direct Current) electricity. The temperature ('T') also affects its performance. Think of it like the power source of the whole system.

Boost Converter:
A boost converter increases voltage using an inductor, switch, diode, and capacitor. The inductor stores energy when the switch (MOSFET/IGBT) is ON. When the switch turns OFF, the inductor releases energy, increasing voltage. The diode allows only the boosted voltage to pass. The capacitor stores the output for a stable DC voltage.

Universal Bridge (Inverter):
This component changes the DC electricity from the boost converter into AC (Alternating Current) electricity. It uses switches that are turned on and off in a specific pattern..

Load:
This represents whatever the solar power system is powering (e.g., a house, a grid). In this model, it looks like a simple resistor, representing a basic electrical load.

Measurement Blocks:
Voltage Measurement: This block measures the voltage at a specific point in the circuit, allowing you to monitor the system's performance. The scopes display the measured values during the simulation.

Capacitor next to the PV array:
The capacitor stabilizes the solar panel's DC voltage output. It smooths out any voltage ripples or fluctuations. This ensures a cleaner and more consistent DC supply. It acts as a temporary energy buffer. This improved voltage quality benefits the boost converter's performance.

PWM generator:
The PWM generator controls the Universal Bridge switches. It sends ON/OFF signals to these switches. By varying the pulse width, it regulates voltage flow. This creates the desired AC output from DC input.

Reinforcement learning (RL)
A type of machine learning where agents learn optimal actions through rewards and penalties in an environment to achieve goals.

Q-Learning
An RL algorithm that helps agents learn the best actions to take in a given state by maximizing cumulative rewards over time.

State, Action, Reward
The core components of RL: State represents the environment, Action is the decision made, and Reward is feedback on the action's effectiveness.
