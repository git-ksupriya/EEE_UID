Reinforcement Learning-based Frequency Control for Renewable Energy Microgrids
These are the blocks we're using

Microgrid Components

1. Wind Turbine

The wind turbine converts wind energy into electrical energy. It generates power based on wind speed and is integrated into the microgrid.

2. PV Array

The photovoltaic (PV) array generates electricity from sunlight. It is a key renewable energy source in the microgrid.

3. Frequency (Phasor)

The frequency phasor block helps analyze the frequency response of the microgrid and monitors frequency deviations and ensures stability.

4. Wind Source

The wind source block models wind speed variations to simulate real-world wind conditions for the microgrid.

5. Universal Bridge

The universal bridge is used for AC-DC and DC-AC power conversion, essential for integrating renewable energy sources.

6. Signal Generator

The signal generator produces test signals for control and monitoring in the microgrid. it is important because RL agent learns from input and this gives input

7. Power Measurement

The power measurement block monitors real-time voltage, current, and power in the microgrid. these measured data sent to controllers to help in decision making

8. Signal Editor

The signal editor allows users to modify and control input signals for simulation experiments. instead of using fixed input like signal generator,the signal lets you to create and edit custom signals before running the simulation

9. 3-Phase Programmable Voltage Source

This block simulates a three-phase AC voltage source with programmable parameters to test different grid conditions. generates AC power with adjustable voltage and frequency.
It simulates grid connections or independent microgrid power sources. Useful for testing voltage stability and system performance under different conditions.

10. Dynamic Load

The dynamic load represents varying loads in the microgrid, helping in analyzing system performance under different conditions.
