Reinforcement Learning-based Frequency Control for Renewable Energy Microgrids

This project focuses on maintaining a stable frequency in a renewable energy-based stand-alone microgrid using Reinforcement Learning (RL). The RL agent intelligently adjusts the inverterâ€™s modulation index to respond to varying loads and disturbances, thereby regulating frequency in real time.

Our base model in MATLAB Simulink consists of a PV source, a boost converter, a single-phase inverter using a Universal Bridge, and variable resistive loads. The inverter is controlled using PWM signals, where the modulation index is the primary input tuned by the RL agent.

Main Goals:

  • Reinforcement Learning-Based Frequency Control
  • Making a base model to verify the performance
  • MATLAB Simulink Model Validation

DC power from a PV source is stepped up using a boost converter and then converted to AC using a single-phase inverter (Universal Bridge).
Dynamic resistive loads are applied to represent real-life load fluctuations.

2. Reinforcement Learning-Based Frequency Control:

An RL agent is trained to maintain the system's output frequency close to 60 Hz by dynamically adjusting the modulation index of the PWM signal fed to the Universal Bridge inverter.

  • State: Includes current frequency deviation
  • Action: Adjusting the modulation index (value between 0 and 1).
  • Reward: High reward when frequency stays near 60 Hz; penalized for deviations.

The RL controller replaces traditional fixed logic by learning optimal control under variable load conditions.

5. MATLAB Simulink Validation:

Our simulation model includes:

  • PV source + Boost Converter (gain + saturation control)
  • Single-phase inverter using Universal Bridge
  • PWM Generator controlled by modulation index
  • LC Filter for output smoothing
  • Three dynamic resistive loads

Performance is evaluated by using scopes to read frequency, camparison with the base model, and RL behavior under disturbances.

What is a Microgrid?

A localized grid system with renewable sources (like solar PV), local loads, and storage or control systems. It can function with or without the main grid.

Problem Addressed:

Stand-alone microgrids face frequency instability due to load and source variability. Our model uses RL to learn adaptive control policies in real-time.

Project Objective:

To stabilize frequency at 60 Hz using a lightweight RL-based controller and validate its efficiency through Simulink modeling and simulation.
